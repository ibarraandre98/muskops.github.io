Mis proyectos

Estos son los proyectos con los que he trabajado a lo largo de mi carrera
SASPRE

Programa capaz de calcular plagas en los cultivos y llevar un control de estos además de las cosechas desarrollado en C# con MySQL.

SASPRE Mobile

Aplicación móvil de SASPRE realiza todo lo anterior en el Smartphone para más facilidad, desarrollada en Ionic con web services PHP y MySQL.

HBuild

Aplicación móvil la cual es una red social para arquitectos y gente involucrada en construcciones, para ver casas, departamentos y edificios, desarrollada en Ionic con Firebase.

Cosas del Tec Mante

Página web enfocada a estudiantes para calificar y puntuar a los maestros del Tecnológico de El Mante, desarrollada con HTML, CSS y PHP.

Talent Top

Aplicación móvil para evento especial de la carrera la cual permite hacer horario para cursos y pláticas, desarrollada en Flutter con web services PHP.

Bugoff

Aplicación web el cual permite organizar todas aquellos bugs y errores de un proyecto, enfocado en testing, desarrollada con Angular y Firebase.

Encud

Aplicación móvil realizar encuestas por parte del sector salud de la región, desarrollada primeramente en Android con Room y despues transferida a Ionic con web services PHP.

VitroMante

Punto de venta para empresa VitroMante la cual permite administrar el inventario, pedidos etc, desarrollado con C# y SQL Server.

Adamant

Compilador capaz de realizar operaciones matemáticas básicas y mostrar resultados basado en C# pero traducido a español, desarrollado en C# con ayuda de la librería Irony.
